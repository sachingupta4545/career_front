# Career Frontend

A modern Next.js frontend for the Career project. This app provides the user interface for authentication, portfolio management, resume upload, projects, and chatbot interaction.

## Features

- User signup and login
- Portfolio profile management
- Resume upload
- Project listing and management
- AI chatbot integration
- Responsive UI with Next.js App Router

## Tech Stack

- Next.js
- React
- TypeScript
- CSS / Tailwind / custom styles
- REST API integration

## Getting Started

### 1. Install dependencies

```bash
npm install

NEXT_PUBLIC_API_URL=http://localhost:8000
npm run dev
npm run dev
npm run build
